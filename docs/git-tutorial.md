# Git Tutorial for Beginners

## Основные понятия:
- **Repository** - хранилище кода
- **Commit** - сохранение изменений
- **Branch** - ветка разработки
- **Merge** - слияние веток
- **Remote** - удаленный репозиторий

## Рабочий процесс:
1. `git pull` - получить свежие изменения
2. `git checkout -b feature/name` - создать ветку
3. Внести изменения в код
4. `git add .` - добавить изменения
5. `git commit -m "message"` - закоммитить
6. `git push -u origin feature/name` - отправить на GitHub
7. Создать Pull Request на GitHub
